# 🎬 VideoHub – A Simple App for Watching Movies

VideoHub is a lightweight web app designed for convenient browsing and watching movies. The project runs entirely in the browser and requires no backend or installation.

---

# ✨ Functions

1. Play videos directly in your browser.
2. A clean HTML-based interface allows for quick access to content.
3. A simple project structure—easy to expand and modify.
4. A static page (HTML + video folder) allows you to host it on GitHub Pages or any static server.
